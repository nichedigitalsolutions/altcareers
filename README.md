# AltCareers

AltCareers is an AI-powered career discovery platform designed to help people explore alternative careers, qualifications, certifications, apprenticeships, and emerging opportunities beyond traditional university pathways.

## Mission

Help people discover realistic, fulfilling career paths through intelligent, personalized guidance.

## Tech Stack

- Flutter
- Supabase
- PostgreSQL
- OpenAI API
